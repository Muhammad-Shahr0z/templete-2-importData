To import data:

Navigate to the dataImport folder and open VS Code or the terminal.
Run the npm install command to install all necessary dependencies.
Add your project ID and token to the importData.js function.
Finally, run the importData.js command, and your data will be uploaded to Sanity.
To set up the schema:

Take the provided schema files and add them to your project.
Paste these files into the schema/types folder.
In the index.js file inside the schema/types folder, import both schema files.



Data import karne ke liye:

dataImport folder me jayein aur VS Code ya terminal open karein.
npm install command chalayein takay saari zaruri dependencies install ho jayen.
importData.js function me apne project ka ID aur token add karein.
Aakhir me node importData.js command chalayein, aur apka data Sanity par upload ho jayega.
Schema set karne ka tareeqa:

Di gayi schema files ko apne project me include karein.
Un files ko schema/types folder me paste karein.
schema/types folder ke andar maujood index.js file me dono schema files ko import karein.
